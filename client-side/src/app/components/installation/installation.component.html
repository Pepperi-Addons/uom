UOM

This module add support in multiple UOMs per item
As part of the installation, the following will take place:

  • The Units Quantity field will be read only and will contain the total in singles.

  • Two new drop down fields and two new Units Quantity fields (with the AQQM prefix) will be added to the
    transaction to allow the user select the required UOM and define the quantity.

Place these fields in the relevant order center and cart views.
Note: You can use both sets (to allow the user order in two UOMs) or only the first one (to allow the user order in one UOM).
